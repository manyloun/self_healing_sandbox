How to use the API

You can interact with the microservice directly via REST calls (e.g. from Postman or another app).

1. Check System Status & Costs

curl -X GET http://192.168.6.51:8100/api/dashboard

2. Run an Analytics Pipeline

Send a POST request to /api/analyze with the required JSON payload.

curl -X POST "http://192.168.6.51:8100/api/analyze?vehicle_type=yellow&month=3&task=Calculate%20average%20fare&provider=anthropic" \
     -H "accept: application/json" \
     -d ""

3. View Execution History

curl -X GET http://192.168.6.51:8100/api/history
